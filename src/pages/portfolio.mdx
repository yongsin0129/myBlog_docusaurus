# 作品集介紹

## 小氣鬼電影追蹤工具
<img alt="作品介紹圖片" src="/img/portfolio/cheapskate-demo.jpg"  width="75%" height="75%" />

### 介紹
使用者可查詢最新首輪、二輪電影，並將感興趣的電影加入自已的收藏清單，方便日後查看。

網站位置 : [小氣鬼電影追蹤工具](https://project-cheapskate-fe.vercel.app/)
