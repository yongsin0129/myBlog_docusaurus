## echo

<details>
  <summary> 

`echo`是一個非常常用的指令，用於在終端上顯示文字或變數的內容。它的基本語法如下：
```
echo [選項] [文字或變數]
```
  </summary>
`echo`指令的主要功能是將指定的文字或變數的內容輸出到終端上。它可以用於顯示固定的文字、變數的值，或者結合其他指令的輸出。

以下是一些常見的用法和選項：

1. 顯示文字：直接在`echo`指令後面加上要顯示的文字，例如：
  ```
  echo Hello, World!
  ```

2. 顯示變數：使用`$`符號加上變數名稱，`echo`會顯示該變數的值，例如：
  ```
  name="John"
  echo $name
  echo $SHELL // 注意 SHELL 需大寫
  ```

3. 轉義字符：可以使用反斜線（`\`）來插入特殊字符，例如換行符（`\n`）：
  ```
  echo -e "Line 1\nLine 2"
  ```

4. 不換行輸出：預設情況下，`echo`會在輸出後換行，如果不想換行，可以使用`-n`選項：
  ```
  echo -n "This is a "
  echo "test"
  ```

5. 輸出到文件：可以使用重定向（`>`）將`echo`的輸出寫入到文件中，例如：
  ```
  echo "Hello, World!" > output.txt
  ```

6. 顯示特殊字符：使用`-e`選項可以解釋特殊字符的轉義序列，例如`\t`代表水平制表符：
  ```
  echo -e "Column 1\tColumn 2"
  ```

這些只是`echo`指令的一些常見用法，它還有其他更多的選項和功能。要獲取更詳細的資訊，可以在終端上執行`man echo`命令來查看`echo`指令的手冊頁面。
</details>

## pwd

<details>
<summary>

在Linux中，`pwd`是一個常用的指令，用於顯示當前工作目錄（Present Working Directory，縮寫為PWD）的路徑。它的語法非常簡單：
```
pwd
```
</summary>  

執行pwd指令後，系統會在終端上顯示出當前工作目錄的絕對路徑。

以下是一些pwd指令的常見用法：

1. 顯示當前工作目錄：執行pwd指令後，終端會輸出當前工作目錄的絕對路徑，例如：

```
pwd
```

2. 使用選項：`pwd`指令通常不需要使用選項，但它支援一些選項來調整輸出的格式。例如，`-P`選項可以解析符號連結，顯示實際的路徑：
```
pwd -P
```
3. 將結果存入變數：可以將`pwd`指令的結果存入變數中，以便在後續的指令中使用。例如：
```
current_dir=$(pwd)
echo "當前目錄：$current_dir"
```
`pwd`指令對於在終端上確定當前所在的目錄非常有用。
它可以在腳本中使用，或者用於查看當前工作目錄以便進行其他操作。請注意，當你切換目錄時，`pwd`指令的輸出也會相應地改變。

</details>


## ls 

<details>
<summary>

`ls` 是 Linux 系統中常用的指令之一，用於列出目錄中的內容。它的語法非常簡單：

```
ls [選項] [目錄]
```
</summary>  

執行 `ls` 指令後，系統會在終端上顯示出指定目錄中的文件和子目錄。

以下是一些 `ls` 指令的常見用法：

1. 列出當前目錄的內容：執行 `ls` 指令後，終端會輸出當前目錄中的文件和子目錄，例如：

```
ls
```

2. 列出指定目錄的內容：可以指定要列出內容的目錄路徑，例如：

```
ls /path/to/directory
```

3. 使用選項：`ls` 指令支援多種選項，用於調整輸出的格式和顯示的詳細程度。例如，`-l` 選項可以以長格式（long format）顯示文件的詳細資訊，包括權限、擁有者、大小等：

```
ls -l
```

4. 列出隱藏文件：使用 `-a` 選項可以顯示包括隱藏文件在內的所有文件和目錄：

```
ls -a
```

5. 列出文件大小：使用 `-s` 選項可以顯示文件的大小，以千字節（KB）為單位：

```
ls -s
```

6. 排序輸出：使用 `-r` 選項可以反向排序輸出，使用 `-t` 選項可以按照修改時間排序輸出。

```
ls -r
ls -t
```

`ls` 指令非常有用，可以幫助你查看目錄中的文件和子目錄，並根據需要調整輸出的格式和排序方式。請注意，如果未指定目錄，則 `ls` 指令將列出當前目錄的內容。


</details>


## mkdir

<details>
<summary>

`mkdir` 是 Linux 系統中常用的指令之一，用於創建新的目錄。它的語法非常簡單：

```
mkdir [選項] 目錄名稱
```
</summary>  

執行 `mkdir` 指令後，系統會在當前目錄下創建一個新的目錄。

以下是一些 `mkdir` 指令的常見用法：

1. 創建單個目錄：執行 `mkdir` 指令後，指定目錄名稱即可創建一個新的目錄，例如：

```
mkdir mydir
```

2. 創建多個目錄：可以同時創建多個目錄，只需在指令中指定多個目錄名稱，例如：

```
mkdir dir1 dir2 dir3
```

3. 創建目錄層次結構：可以使用 `-p` 選項創建多層次的目錄結構，如果上層目錄不存在，則會自動創建，例如：

```
mkdir -p path/to/mydir
```

4. 創建目錄並設置權限：可以使用 `-m` 選項指定新創建目錄的權限，例如：

```
mkdir -m 755 mydir
```

上述指令將創建一個名為 `mydir` 的目錄，並將其權限設置為 `755`。

`mkdir` 指令非常有用，可以快速創建新的目錄，並根據需要創建多層次的目錄結構。請注意，如果指定的目錄已經存在，則 `mkdir` 指令將報錯。

</details>


## cd

<details>
<summary>

`cd` 是 Linux 系統中常用的指令之一，用於更改當前工作目錄。它的語法非常簡單：

```
cd [目錄]
```
</summary>  

執行 `cd` 指令後，系統會將當前工作目錄切換到指定的目錄。

以下是一些 `cd` 指令的常見用法：

1. 切換到指定目錄：執行 `cd` 指令後，指定要切換到的目錄路徑即可，例如：

```
cd /path/to/directory
```

2. 切換到上層目錄：可以使用 `..` 表示上層目錄，例如：

```
cd ..
```

這將將當前工作目錄切換到上層目錄。

3. 切換到家目錄：可以使用 `~` 表示家目錄（即當前用戶的主目錄），例如：

```
cd ~
```

這將將當前工作目錄切換到家目錄。

4. 切換到先前目錄：可以使用 `-` 表示先前所在的目錄，例如：

```
cd -
```

這將將當前工作目錄切換到先前所在的目錄。

`cd` 指令非常有用，可以快速切換工作目錄，方便在不同的目錄之間進行操作。請注意，如果指定的目錄不存在或無法訪問，則 `cd` 指令將報錯。

</details>

## touch

<details>
<summary>

`touch` 是一個常用的指令，用於創建空白文件或更新現有文件的時間戳。它的語法非常簡單：

```
touch [選項] 文件名稱
```
</summary>  

執行 `touch` 指令後，系統會根據指定的文件名稱創建一個新的空白文件，或者更新現有文件的存取和修改時間。

以下是一些 `touch` 指令的常見用法：

1. 創建新文件：執行 `touch` 指令後，指定要創建的文件名稱即可，例如：

```
touch myfile.txt
```

這將創建一個名為 `myfile.txt` 的空白文件。

2. 更新文件時間戳：如果指定的文件已經存在，執行 `touch` 指令將更新該文件的存取和修改時間，例如：

```
touch myfile.txt
```

這將更新 `myfile.txt` 文件的時間戳。

3. 同時創建多個文件：可以同時創建多個文件，只需在指令中指定多個文件名稱，例如：

```
touch file1.txt file2.txt file3.txt
```

這將同時創建 `file1.txt`、`file2.txt` 和 `file3.txt` 這三個空白文件。

`touch` 指令還有其他一些選項可以使用，例如可以使用 `-c` 選項在文件不存在時不創建文件，使用 `-r` 選項從現有文件複製時間戳等。可以通過執行 `man touch` 命令來查看 `touch` 指令的詳細用法和選項。

`touch` 指令是一個方便的工具，可以用於創建空白文件或更新文件的時間戳。

</details>

## rm

<details>
<summary>

`rm` 是 Linux 系統中常用的指令之一，用於刪除文件或目錄。它的語法非常簡單：

```
rm [選項] 文件或目錄
```
</summary>  

執行 `rm` 指令後，系統會刪除指定的文件或目錄。

以下是一些 `rm` 指令的常見用法：

1. 刪除文件：執行 `rm` 指令後，指定要刪除的文件名稱即可，例如：

```
rm myfile.txt
```

這將刪除名為 `myfile.txt` 的文件。

2. 刪除目錄：如果要刪除一個目錄及其內容，可以使用 `-r` 或 `--recursive` 選項，例如：

```
rm -r mydirectory
```

這將刪除名為 `mydirectory` 的目錄及其內容。

3. 確認刪除：如果希望在刪除文件或目錄之前進行確認，可以使用 `-i` 或 `--interactive` 選項，例如：

```
rm -i myfile.txt
```

這將在刪除 `myfile.txt` 文件之前提示確認。

4. 強制刪除：有時候可能會遇到無法刪除的文件或目錄，可以使用 `-f` 或 `--force` 選項強制刪除，例如：

```
rm -rf mydirectory
```

這將強制刪除 `mydirectory` 目錄及其內容，而無需進行確認。

請注意，`rm` 指令刪除的文件或目錄將無法恢復，請在使用 `rm` 指令時謹慎操作，確保要刪除的內容是正確的。

`rm` 指令還有其他一些選項可以使用，例如可以使用 `-v` 或 `--verbose` 選項顯示詳細的刪除信息，使用 `-i` 或 `--interactive` 選項逐個確認刪除等。可以通過執行 `man rm` 命令來查看 `rm` 指令的詳細用法和選項。

`rm` 指令是一個強大的工具，可以用於刪除文件或目錄。請在使用時謹慎操作，以免不小心刪除重要的文件或目錄。

</details>

## cp

<details>
<summary>

`cp` 是 Linux 系統中常用的指令之一，用於複製文件或目錄。它的語法非常簡單：

```
cp [選項] 源文件 目標文件
```
</summary>  

執行 `cp` 指令後，系統會將源文件複製到目標文件。

以下是一些 `cp` 指令的常見用法：

1. 複製文件：執行 `cp` 指令後，指定要複製的源文件和目標文件即可，例如：

```
cp myfile.txt mycopy.txt
```

這將複製名為 `myfile.txt` 的文件到 `mycopy.txt`。

2. 複製目錄：如果要複製一個目錄及其內容，可以使用 `-r` 或 `--recursive` 選項，例如：

```
cp -r sourcedir targetdir
```

這將複製名為 `sourcedir` 的目錄及其內容到 `targetdir`。

3. 複製多個文件：可以同時複製多個文件，只需在指令中指定多個源文件，並指定目標目錄，例如：

```
cp file1.txt file2.txt targetdir
```

這將複製 `file1.txt` 和 `file2.txt` 兩個文件到 `targetdir` 目錄。

4. 覆蓋已存在的文件：如果目標文件已經存在，默認情況下 `cp` 指令會提示是否覆蓋，可以使用 `-f` 或 `--force` 選項強制覆蓋，例如：

```
cp -f myfile.txt mycopy.txt
```

這將強制將 `myfile.txt` 複製到 `mycopy.txt`，即使 `mycopy.txt` 已經存在。

`cp` 指令還有其他一些選項可以使用，例如可以使用 `-v` 或 `--verbose` 選項顯示詳細的複製信息，使用 `-i` 或 `--interactive` 選項逐個確認複製等。可以通過執行 `man cp` 命令來查看 `cp` 指令的詳細用法和選項。

`cp` 指令是一個非常實用的工具，可以用於複製文件或目錄。請確保在使用時指定正確的源文件和目標位置，以免不小心覆蓋或刪除重要的內容。

</details>



## mv

<details>
<summary>

`mv` 是 Linux 系統中常用的指令之一，用於移動文件或目錄，或者將文件或目錄重新命名。它的語法非常簡單：

```
mv [選項] 源文件 目標位置
```
</summary>  

執行 `mv` 指令後，系統會將源文件移動到目標位置，或者將文件重新命名。

以下是一些 `mv` 指令的常見用法：

1. 移動文件：執行 `mv` 指令後，指定要移動的源文件和目標位置即可，例如：

```
mv myfile.txt myfolder/
```

這將把名為 `myfile.txt` 的文件移動到 `myfolder/` 目錄中。

2. 移動目錄：如果要移動一個目錄及其內容，可以使用 `-r` 或 `--recursive` 選項，例如：

```
mv sourcedir/ targetdir/
```

這將把名為 `sourcedir/` 的目錄及其內容移動到 `targetdir/`。

3. 重新命名文件：可以使用 `mv` 指令將文件重新命名，只需指定源文件和目標位置，例如：

```
mv oldname.txt newname.txt
```

這將把名為 `oldname.txt` 的文件重新命名為 `newname.txt`。

4. 覆蓋已存在的文件：如果目標位置已經存在同名的文件或目錄，默認情況下 `mv` 指令會提示是否覆蓋，可以使用 `-f` 或 `--force` 選項強制覆蓋，例如：

```
mv -f myfile.txt myfolder/
```

這將強制將 `myfile.txt` 移動到 `myfolder/`，即使 `myfolder/` 中已經存在同名的文件。

`mv` 指令還有其他一些選項可以使用，例如可以使用 `-v` 或 `--verbose` 選項顯示詳細的移動信息，使用 `-i` 或 `--interactive` 選項逐個確認移動等。可以通過執行 `man mv` 命令來查看 `mv` 指令的詳細用法和選項。

`mv` 指令是一個非常實用的工具，可以用於移動文件或目錄，或者將文件或目錄重新命名。請確保在使用時指定正確的源文件和目標位置，以免不小心覆蓋或刪除重要的內容。

</details>


## grep

<details>
<summary>

`grep` 是 Linux 系統中常用的文本搜索工具，用於在文件或文本中查找符合指定模式的行。它的語法非常簡單：

```
grep [選項] 模式 文件
```
</summary>  

執行 `grep` 指令後，系統會在指定的文件中搜索包含指定模式的行，並將匹配的行輸出到終端。

以下是一些 `grep` 指令的常見用法：

1. 基本搜索：執行 `grep` 指令後，指定要搜索的模式和文件即可，例如：

```
grep "keyword" myfile.txt
```

這將在 `myfile.txt` 文件中搜索包含 "keyword" 的行。

2. 正則表達式搜索：`grep` 支持使用正則表達式進行更靈活的搜索。例如，要搜索以 "pattern" 開頭的行，可以使用正則表達式 `^pattern`，例如：

```
grep "^pattern" myfile.txt
```

這將在 `myfile.txt` 文件中搜索以 "pattern" 開頭的行。

3. 忽略大小寫：默認情況下，`grep` 是區分大小寫的。如果希望進行不區分大小寫的搜索，可以使用 `-i` 或 `--ignore-case` 選項，例如：

```
grep -i "keyword" myfile.txt
```

這將在 `myfile.txt` 文件中不區分大小寫地搜索 "keyword"。

4. 遞歸搜索：如果要在目錄及其子目錄中進行搜索，可以使用 `-r` 或 `--recursive` 選項，例如：

```
grep -r "keyword" mydir/
```

這將在 `mydir/` 目錄及其子目錄中搜索包含 "keyword" 的行。

`grep` 指令還有其他一些選項可以使用，例如可以使用 `-v` 或 `--invert-match` 選項顯示不匹配的行，使用 `-n` 或 `--line-number` 選項顯示行號等。可以通過執行 `man grep` 命令來查看 `grep` 指令的詳細用法和選項。

`grep` 是一個非常強大的文本搜索工具，可以根據指定的模式在文件或文本中進行搜索。它在處理大型文件或需要進行高級搜索的場景下非常有用。

</details>


## cat

<details>
<summary>

`cat` 是 Linux 系統中常用的指令之一，用於連接文件並將它們的內容輸出到終端或其他文件。它的語法非常簡單：

```
cat [選項] 文件...
```
</summary>  

執行 `cat` 指令後，系統會按照指定的順序連接並輸出指定文件的內容。

以下是一些 `cat` 指令的常見用法：

1. 查看文件內容：執行 `cat` 指令後，指定要查看的文件即可，例如：

```
cat myfile.txt
```

這將輸出 `myfile.txt` 文件的內容到終端。

2. 連接多個文件：可以指定多個文件名作為 `cat` 指令的參數，它們的內容將按照指定的順序連接起來，例如：

```
cat file1.txt file2.txt
```

這將連接 `file1.txt` 和 `file2.txt` 的內容並輸出到終端。

3. 將內容輸出到文件：可以使用重定向符號 `>` 將 `cat` 指令的輸出重定向到一個新文件中，例如：

```
cat myfile.txt > newfile.txt
```

這將把 `myfile.txt` 的內容輸出到 `newfile.txt` 中，如果 `newfile.txt` 已經存在，則會被覆蓋。

4. 顯示行號：可以使用 `-n` 或 `--number` 選項顯示輸出的行號，例如：

```
cat -n myfile.txt
```

這將輸出 `myfile.txt` 的內容並在每一行前面顯示行號。

`cat` 指令還有其他一些選項可以使用，例如可以使用 `-b` 或 `--number-nonblank` 選項僅對非空行顯示行號，使用 `-s` 或 `--squeeze-blank` 選項壓縮連續的空行等。可以通過執行 `man cat` 命令來查看 `cat` 指令的詳細用法和選項。

`cat` 指令是一個非常實用的工具，可以用於查看文件內容、連接文件以及將內容輸出到其他文件。請注意在使用時確保指定正確的文件名和順序，以免不小心覆蓋或丟失重要的內容。

</details>
