## echo

<details>
  <summary> 

`echo`是一個非常常用的指令，用於在終端上顯示文字或變數的內容。它的基本語法如下：
```
echo [選項] [文字或變數]
``` 

  </summary>
`echo`指令的主要功能是將指定的文字或變數的內容輸出到終端上。它可以用於顯示固定的文字、變數的值，或者結合其他指令的輸出。

以下是一些常見的用法和選項：

1. 顯示文字：直接在`echo`指令後面加上要顯示的文字，例如：
  ```
  echo Hello, World!
  ```

2. 顯示變數：使用`$`符號加上變數名稱，`echo`會顯示該變數的值，例如：
  ```
  name="John"
  echo $name
  echo $SHELL // 注意 SHELL 需大寫
  ```

3. 轉義字符：可以使用反斜線（`\`）來插入特殊字符，例如換行符（`\n`）：
  ```
  echo -e "Line 1\nLine 2"
  ```

4. 不換行輸出：預設情況下，`echo`會在輸出後換行，如果不想換行，可以使用`-n`選項：
  ```
  echo -n "This is a "
  echo "test"
  ```

5. 輸出到文件：可以使用重定向（`>`）將`echo`的輸出寫入到文件中，例如：
  ```
  echo "Hello, World!" > output.txt
  ```

6. 顯示特殊字符：使用`-e`選項可以解釋特殊字符的轉義序列，例如`\t`代表水平制表符：
  ```
  echo -e "Column 1\tColumn 2"
  ```

這些只是`echo`指令的一些常見用法，它還有其他更多的選項和功能。要獲取更詳細的資訊，可以在終端上執行`man echo`命令來查看`echo`指令的手冊頁面。
</details>
