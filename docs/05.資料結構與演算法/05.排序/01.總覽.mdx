## 表格       
  | 演算法 | 平均時間複雜度 | 最壞時間複雜度 | 最好時間複雜度 | 空間複雜度 | 是否穩定排序 |
  | --- | --- | --- | --- | --- | --- |
  | 插入排序 | O(n^2) | O(n^2) | O(n) | O(1) | 穩定 |
  | 選擇排序 | O(n^2) | O(n^2) | O(n^2) | O(1) | 不穩定 |
  | 泡沫排序 | O(n^2) | O(n^2) | O(n) | O(1) | 穩定 |
  | 快速排序 | O(n log n) | O(n^2) | O(n log n) | O(log n)  依實現方式不同而不同 | 不穩定 |
  | 合併排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 穩定 |
  | 希爾排序
  (插入排序改良) |  | O(n^2) | O(n log n) | O(1) |  |
  | 堆排序
  (選擇排序改良) | O(n log n) | O(n log n) | O(n log n) | O(1) | 不穩定 |
