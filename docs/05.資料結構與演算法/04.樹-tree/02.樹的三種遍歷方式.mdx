## Binary Tree Traversal 二元樹遍歷

前序 (preorder), 中序 (inorder) 和後序 (postorder) 是指遍歷二元樹 (binary tree) 時，父節點相對於左右節點的順序。假設二元樹如下：
```
    4
   / \
  2   6
 / \ / \
1  3 5  7
```

則三種遍歷的順序依序為：

- 前序 (preorder): 中 -> 左 -> 右，4213657
- 中序 (inorder): 左 -> 中 -> 右，1234567。
- 後序 (postorder): 左 -> 右 -> 中，1325764

注意：對二元搜尋樹 (binary search tree, BST) 做 inorder traversal 就是由小到大依序遍歷。

## 實作方式 - JS程式碼    

  以下是 js 的二元樹節點的實現：
  ```
  function Node(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
  
  ```
  
  接下來分別是這三種遍歷方式的實現：
  
  1. Preorder Traversal (先序遍歷)
  
  ```
  function preorderTraversal(node) {
    if (node !== null) {
      console.log(node.value);
      preorderTraversal(node.left);
      preorderTraversal(node.right);
    }
  }
  
  ```
  
  1. Inorder Traversal (中序遍歷)
  
  ```
  function inorderTraversal(node) {
    if (node !== null) {
      inorderTraversal(node.left);
      console.log(node.value);
      inorderTraversal(node.right);
    }
  }
  
  ```
  
  1. Postorder Traversal (後序遍歷)
  
  ```
  function postorderTraversal(node) {
    if (node !== null) {
      postorderTraversal(node.left);
      postorderTraversal(node.right);
      console.log(node.value);
    }
  }
  
  ```
  
  注意：以上的程式碼都是基於二元樹的概念，有些類似於遞迴的用法，需要了解其原理和運作方式，才能更好地理解程式碼。
