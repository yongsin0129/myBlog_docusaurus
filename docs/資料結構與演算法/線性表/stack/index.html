stack

繁中「堆疊」，簡中「栈」。像疊盤子，顛倒資料前後順序。

Array 和 List 皆可實作。

插入、刪除需時 O(1) 。搜尋需時 O(N) 。

堆疊有反轉的性質、有括號對應的性質、有遞迴與疊代的性質。

Stack（堆疊）是一種常用的資料結構，類似於生活中的棧盤子，後進先出（Last-In-First-Out，LIFO）的原則。

它的基本操作有 push（壓入）、pop（彈出）和 top（獲取堆疊頂部元素），可以用於模擬瀏覽器的前進和後退、計算機的表達式求值等。

堆疊的 push 操作會將元素壓入堆疊頂部，使其成為堆疊的新的頂部元素；pop 操作會將堆疊頂部的元素彈出，使其下面的元素成為新的頂部元素；top 操作則是獲取堆疊頂部的元素，但是不彈出。

堆疊的實現可以使用數組或鏈表，使用數組實現的堆疊稱為數組堆疊，使用鏈表實現的堆疊稱為鏈式堆疊。

在使用堆疊時需要注意堆疊是否已滿或者是否已空，以避免出現溢出或下溢的情況。

在計算機科學中，堆疊的應用非常廣泛，包括編譯器的語法分析、操作系統的進程管理、網頁瀏覽器的前進和後退等。
